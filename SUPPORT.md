# Support

## Security issues
Report vulnerabilities through the process in `SECURITY.md`.

## Usage questions and bug reports
Open a GitHub issue with:
- runtime and version information
- minimal CSS input reproducer
- expected vs actual result
- command outputs (`npm run eval:ci`, `npm run eval:release`, `npm run eval:hard-gate`)

## Scope reminders
This project is a deterministic CSS parser with selector/query utilities. It is not a browser rendering engine.
